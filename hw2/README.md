## Environment Setup
- Python 3.9
```bash
conda create -n adl-hw2 python=3.9
conda activate adl-hw2
```
- PyTorch 2.1.0
```bash
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
```
- Other dependencies
```bash
pip install -r requirements.txt
```
